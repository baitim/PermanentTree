## How to run

1. Clone <br>
    <code>git clone https://github.com/baitim/PermanentTree.git</code>

2. Go to folder <br>
    <code>cd PermanentTree</code>

3. Build <br>
    <code>cmake . -B build ; cmake --build build</code>

4. Run <br>
    <code>./build/src/perm_tree</code> in terminal <br>
